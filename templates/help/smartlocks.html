What smart locks track

Smart locks are keypad/electronic locks installed at properties. KBM stores each lock's label, current code, the property and unit it's installed at, notes, and photos (e.g. of the install or the lock model).

Adding a smart lock

  1. Go to Smart Locks → Add.
  2. Enter a label and the code, and link it to a property/unit.

Codes

When a code changes (turnover, contractor finished), edit the lock and enter the new code. Changes are recorded in the activity log, so you can see when a code was last rotated and by whom.

Photos

Open a lock's page to upload photos with captions. Photos are validated and stored safely — handy for remembering which unit door a lock is on.

Import / export

Bulk-load locks from a spreadsheet with Import, or download the current list with Export — see the Importing Data topic for the wizard walkthrough.
