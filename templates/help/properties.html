Properties

Properties are the addresses your inventory belongs to. Each property has a name, type (single family, multi-family, …), full address, and optional notes. The property page shows everything assigned to it: keys, lockboxes, signs, and smart locks.

Adding a property

  1. Go to Properties → Add Property.
  2. Name and street address are required; city/state/ZIP help searches and exports.

Units

Multi-family properties can have units (e.g. "Unit 1A"). Add them from the property's page. Keys and smart locks can be linked to a specific unit, so "which key opens 12 Maple St Unit 3B?" is one search away.

Linking inventory

  • When adding a key, pick the property (and unit) it opens.
  • Lockboxes are assigned to a property from their detail page after creation.
  • Signs get assigned to a property when deployed.
Deleting a unit requires an admin, and inventory linked to it keeps working — the link is simply cleared.

Importing many at once

Use Import (properties) or Import Units from the Properties page to load a spreadsheet — see the Importing Data topic.
