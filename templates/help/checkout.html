The fast lane

The Check In/Out page is built for speed at the front desk: scan a label, hit Enter, done. It works for lockboxes, keys, and signs.

Finding the item

  • Scan the QR/barcode on the item's printed label, or
  • Type its ID (e.g. KA042) or start typing the label/address — autocomplete suggests matches as you type.
  • You can also search by person to see everything a name currently has checked out or assigned — handy when someone returns a pile of keys.

Checking out

  1. Pull up the item. Its details and current status appear.
  2. Fill in who's taking it — pick an existing contact (they'll get an email confirmation if notifications are on) or just type a name.
  3. For keys, set how many copies.
  4. Set the purpose and expected return date — this drives the overdue/upcoming reports and email reminders.

Checking in

Pull up the item and choose check in. For keys, enter how many copies came back. For lockboxes you can set a new code at the same time.

Tip: after every checkout you're offered a printable receipt. If the person wants one later, find it under Receipts.
