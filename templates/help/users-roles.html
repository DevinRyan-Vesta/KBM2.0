Roles

  • Owner / Admin — full control: manage users, edit and delete inventory, run audits, change settings, view activity logs.
  • User / Staff — day-to-day work: add inventory, check items in and out, manage properties and contacts.
  • Agent — limited access for outside agents.

Adding a user (admins)

  1. Go to Administration → Users → Add User.
  2. Enter name, email, role, and a PIN.

PINs

  • Everyone logs in with just their PIN, so each PIN must be unique within your company — KBM enforces this.
  • Minimum 4 characters; longer is safer. Avoid obvious ones (1234, birth years).
  • Users can change their own PIN from Profile; admins can reset anyone's from the Users page.
Because activity is recorded per user, shared PINs defeat the audit trail. One person, one PIN.

Deactivating someone

When staff leave, deactivate their user rather than deleting it — their history stays intact but they can no longer log in.

Activity Logs

Administration → Activity Logs shows every meaningful action — logins, checkouts, edits, deletions, settings changes — with who, what, and when. Filter it or export to CSV.
